---
title: Docker
---

You can use Lumier through Docker:

### Run a macOS VM (ephemeral)
```bash
docker run -it --rm \
  --name macos-vm \
  -p 8006:8006 \
  -e VM_NAME=macos-vm \
  -e VERSION=ghcr.io/trycua/macos-sequoia-cua:latest \
  -e CPU_CORES=4 \
  -e RAM_SIZE=8192 \
  trycua/lumier:latest
```
Access the VM in your browser at [http://localhost:8006](http://localhost:8006).

### Persistent Storage
```bash
mkdir -p storage

docker run -it --rm \
  --name lumier-vm \
  -p 8006:8006 \
  -v $(pwd)/storage:/storage \
  -e VM_NAME=lumier-vm \
  -e VERSION=ghcr.io/trycua/macos-sequoia-cua:latest \
  -e CPU_CORES=4 \
  -e RAM_SIZE=8192 \
  -e HOST_STORAGE_PATH=$(pwd)/storage \
  trycua/lumier:latest
```

### Shared Folder
```bash
mkdir -p shared

docker run -it --rm \
  --name lumier-vm \
  -p 8006:8006 \
  -v $(pwd)/storage:/storage \
  -v $(pwd)/shared:/shared \
  -e VM_NAME=lumier-vm \
  -e VERSION=ghcr.io/trycua/macos-sequoia-cua:latest \
  -e CPU_CORES=4 \
  -e RAM_SIZE=8192 \
  -e HOST_STORAGE_PATH=$(pwd)/storage \
  -e HOST_SHARED_PATH=$(pwd)/shared \
  trycua/lumier:latest
```
